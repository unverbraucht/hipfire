# hipfire

Rust-native ML inference engine for AMD RDNA GPUs. Targets Qwen 3.5/3.6 dense + MoE families with custom HIP kernels (HFQ4/MQ4 quant, FP16 WMMA, dp4a, Lloyd-Max codebooks).

## Build

```
cargo build --release --features deltanet -p hipfire-runtime --example daemon
```

## Test

Run `./scripts/coherence-gate.sh` for the short coherence battery (~3-4 min). For multi-GPU PP work, also run `./scripts/pp-gate.sh`.
